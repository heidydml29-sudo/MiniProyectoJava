Sistema de Refugio de Animales (Java)
Descripción: Este proyecto es una aplicación de consola desarrollada en Java que simula la gestión básica de un refugio de animales.
El sistema permite registrar perros y gatos, consultar animales almacenados, cambiar su estado y ejecutar el sonido característico de cada animal.
El objetivo del proyecto es aplicar conceptos fundamentales de Programación Orientada a Objetos (POO).

Objetivos del Proyecto: 
* Aplicar clases y objetos
* Implementar encapsulamiento
* Utilizar herencia
* Aplicar polimorfismo
* Gestionar datos con ArrayList
* Manejar errores con try-catch
* Crear un menú interactivo en consola

Estructura del Proyecto

RefugioAnimales
 Animal.java
 Perro.java
Gato.java
Refugio.java
Main.java
Descripción de las clases

Animal.java
Clase base del sistema.
Contiene los atributos comunes de los animales:
  * id
  * nombre
  * edad
  * estado

Perro.java
* Hereda de la clase Animal.
* Implementa el sonido característico del perro.

Gato.java
* Hereda de la clase Animal.
* Implementa el sonido característico del gato.

Refugio.java
* Gestiona la lista de animales utilizando ArrayList.
* Permite registrar, buscar, listar y actualizar animales.

Main.java
* Contiene el menú principal del sistema.
* Permite interactuar con el usuario mediante consola.


Funcionalidades
El sistema permite:

Registrar perros
Registrar gatos
Mostrar todos los animales
Buscar animal por ID
Cambiar estado del animal
Ejecutar sonido del animal
Salir del sistema
